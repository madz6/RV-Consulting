# RV-Consulting

GiveAGo project #1 — **Pricing Strategy Workbench**

This repository contains the complete product-definition package for a human-led, evidence-grounded pricing strategy system (~23,000 words across nine canonical documents). No product implementation has been started yet.

**Start here:** [`docs/product/README.md`](docs/product/README.md) — the product documentation index.

## Central product decision

A **consultant-led evidence-to-decision system** for new-offer monetisation under sparse historical data — not a price optimiser or autonomous consultancy.

## Scope levels

| Level | Purpose |
|---|---|
| **Synthetic hackathon demonstrator** | Thin end-to-end slice: evidence synthesis, one human-authored strategy plus an AI-assisted branch, one case-specific model, one review-ready decision package, and one assumption-change demonstration |
| **Securely operated concierge design-partner pilot** | Human-operated engagement with production-grade isolation and review controls |
| **Native product pilot** | Full workflow, roles, gates and reproducible modelling in product form |
