# RV-Consulting

GiveAGo project #1 — **Pricing Strategy Workbench**

This repository contains the product definition for a human-led, evidence-grounded pricing strategy system.

Start with [`docs/product/README.md`](docs/product/README.md). It is the canonical navigation page for humans and Codex.

## Working thesis

Pricing Strategy Workbench turns an ambiguous commercial pricing question into a researched, model-tested and approval-ready decision package. It is designed to help consultants create strategy, not merely analyse transactions or automate live prices.
